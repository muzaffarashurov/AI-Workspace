# AI-Orchestrator

## Цель

AI-Orchestrator — центральный исполнительный модуль AI-Workspace.

Он принимает команды пользователя, анализирует их, выбирает подходящего AI, управляет задачами, контролирует выполнение и возвращает результат.

---

## Главные обязанности

- принимать текстовые команды;
- принимать голосовые команды;
- переводить запросы на английский язык при необходимости;
- выбирать наиболее подходящий AI;
- управлять несколькими AI одновременно;
- сохранять общий контекст задач;
- автоматически переключать выполнение между AI;
- запускать локальные программы;
- взаимодействовать с Python, GitHub, Telegram, Excel и другими сервисами.

---

## Главная идея

Пользователь никогда не обращается напрямую к отдельному AI.

Пользователь общается только с AI-Orchestrator.

---

# Vision

AI-Workspace — это персональная интеллектуальная операционная система инженера.

Основная цель проекта — создать цифрового помощника, который способен понимать естественный язык, самостоятельно планировать выполнение задач, управлять несколькими AI-моделями и автоматизировать инженерную работу.

Пользователь общается только с AI-Orchestrator.

Все остальные AI, программы и сервисы являются исполнителями.

---

# Long-Term Goal

Через несколько лет пользователь сможет дать голосовую команду:

> "Подготовь отчёт по новым дефектам, создай презентацию, отправь накладные и начни разработку новой программы."

AI-Orchestrator самостоятельно:

- разобьёт задачу на этапы;
- назначит исполнителей;
- сохранит контекст;
- проверит результат;
- предложит пользователю готовое решение.
