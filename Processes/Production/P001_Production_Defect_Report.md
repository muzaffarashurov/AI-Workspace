# P001 – Production Defect Report

## Purpose

Подготовить ежедневный отчёт по производственным дефектам для руководства.

---

## Schedule

Понедельник, среда, пятница

08:00–09:15

Совещание в 10:00

---

## Trigger

Наступление рабочего дня согласно графику.

---

## Input

- Данные по дефектам
- Производственные журналы
- Excel
- Информация от производственных линий

---

## Production Lines

- Резка и предварительная обработка
- Ламинация
- Термообработка
- Термопласт

---

## Workflow

1. Получить данные.
2. Проверить корректность.
3. Заполнить Excel.
4. Проверить итоговые показатели.
5. Подготовить отчёт.
6. Передать руководителю.

---

## Output

Готовый отчёт для совещания.

---

## Used Software

- Microsoft Excel
- ChatGPT (при необходимости)

- Автоматическое заполнение Excel
- Построение диаграмм
- AI-анализ причин возникновения дефектов
