# WD-005 – Рабочий день: Пятница

## Purpose

Подготовить данные по производственному браку к совещанию по качеству
и организовать остаток дня перед выходными.

---

## Schedule

08:00–09:15 — подготовка Fast Response Board
10:00 — совещание по качеству
В течение дня — накладные, входящий груз, контроль дефектов

---

## Trigger

Начало рабочего дня (пятница).

---

## Input

- Данные по производственным линиям за предыдущий день
- Накладные, поступившие с утра
- Информация о запланированных поставках на выходные

---

## Workflow

1. Заполнить Fast Response Board.
2. Собрать данные по линии резки и предварительной обработки.
3. Собрать данные по ламинации.
4. Собрать данные по термообработке.
5. Собрать данные по термопласту.
6. Проверить корректность собранных данных.
7. Подготовить отчёт к совещанию по качеству.
8. Провести совещание по качеству (10:00).
9. Проверить новые накладные.
10. Отправить накладные через 1С.
11. Проконтролировать входящие грузы.
12. Проверить производственные дефекты.
13. Проверить, что все задачи недели закрыты перед выходными.

---

## Output

Отчёт готов к совещанию в 10:00. Накладные обработаны, входящий груз под контролем,
неделя закрыта.
