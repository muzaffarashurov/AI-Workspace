# AI-Workspace

AI-Workspace — это персональная инженерная база знаний и система управления рабочими процессами.

## Структура

- Docs — документация проекта
- Processes — библиотека рабочих процессов
- Prompts — промпты для AI
- Templates — шаблоны документов

## Цель

Стандартизировать рабочие процессы, сохранить знания и постепенно автоматизировать повторяющиеся задачи.
